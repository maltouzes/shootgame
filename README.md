# shootgame
A touch game, you must kill the birds.

Tested on Linux, but should work on Windows, Mac and BSD too.
Very funnier on Android!

# Installation
Prerequisites:
* Python 3.3 (tested with Python3.5)
* Kivy 1.9 (tested with Kivy 1.10)
